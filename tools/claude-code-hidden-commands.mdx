## 背景

Claude Code 的更新速度极快，很多实用功能藏在更新日志甚至开发团队的 Twitter 随口一提里。以下整理了 10 个经过实际验证的隐藏命令和快捷键。

---

## 一、/btw — 不污染上下文的旁路提问

> 2026 年 3 月 11 日上线

在 Claude 正在执行任务时，输入 `/btw` 加空格，可以插入一个**不会进入对话历史**的临时问题。

**解决的痛点**：以前让 Claude 重构大模块时，中途想问个问题（比如"测试文件在哪个目录"），一问就中断了任务，还污染了上下文，导致 Claude 跑偏。

**用法**：
1. Claude 执行任务中，输入 `/btw 你的问题`
2. Claude 在并行状态下回答，**不中断当前任务**
3. 看完回答后按空格或回车，回答直接消失
4. 对话历史干干净净，几乎不费 token（复用了当前提示缓存）

---

## 二、/rewind — 代码和对话分别回退

> 快捷键：连按两次 `Esc`

相当于代码编辑器里的 `Ctrl+Z`，但更强大——代码和对话可以**分别回退**。

**用法**：输入 `/rewind` 后会弹出历史列表，点击想回退到的节点，出现四个选项：

| 选项 | 效果 |
|------|------|
| 回退代码和对话 | 完全回到那个时间点 |
| 回退对话，保留代码 | 代码改动留着，对话清掉 |
| 回退代码，保留对话 | 代码还原，但 Claude 仍记得之前讨论内容 |
| 从该点压缩对话 | 释放上下文窗口空间 |

**最佳实践**：让 Claude 试一种新方案，不行的话——代码回退、对话保留。这样 Claude 还记得这条路不通，可以直接换方向，不用重新解释需求。

## 三、/insights — AI 反向观察你的使用习惯

输入 `/insights`，Claude Code 会生成一份 **HTML 报告**，分析你过去一个月的使用习惯：

- 最常用的命令
- 重复性的操作模式
- 推荐的自定义命令和 Skills
- 翻车现场回放和改进建议

建议每月跑一次，会让你重新认识自己的编码习惯。

## 四、/model opusplan — Pro 用户的省量神器

这是一个**真正的隐藏命令**——在 `/model` 的模型列表里看不到它。

```bash
/model opusplan
```

**效果**：需要复杂推理时自动用 Opus 4.6 做规划，具体写代码时切换到 Sonnet 4.6 执行。

**适合谁**：$20/月 Pro 订阅用户。Opus 额度有限，全程 Opus 容易被限速。规划需要深度思考用 Opus，写代码 Sonnet 完全够用且更快——两全其美。

---

## 五、/simplify — 三合一代码审查

> 2026 年 2 月底集成为内置 Skill

输入 `/simplify` 后，Claude Code 同时启动**三个并行 Agent**，分别从以下角度审查改动：

1. **代码复用** — 是否有重复逻辑可以抽取
2. **代码质量** — 风格、可读性、多余 import
3. **运行效率** — 性能优化空间

**使用建议**：每次和 Claude Code 对话多轮、写了几个大功能后，顺手跑一遍。AI 写的代码经常有微妙的冗余，`/simplify` 基本都能挑出来。

## 六、/branch — 对话分叉

以前叫 `/fork`，现在改名为 `/branch`（输入 `/fork` 仍可自动跳转）。

把当前对话分叉出一个新会话，**原会话不受影响**。

**使用场景**：Claude 帮你梳理完一个方案思路后，想试两种不同的实现方式——`/branch` 一下，两个会话各走一边，最后挑效果好的。

> `/rewind` 是后悔药，`/branch` 是平行宇宙。

---

## 七、/loop — 定时循环执行

让 Claude 定时重复执行某个任务。

```bash
/loop 5m 检查一下部署状态
```

- 默认间隔 10 分钟
- 结果直接在对话上下文里，Claude 可以基于结果做后续判断
- 创建 3 天后自动过期并自我删除，防止被遗忘的循环一直跑

---

## 八、/remote-control（/rc）— 手机遥控

输入 `/rc` 或 `/remote-control`，生成一个 URL。手机打开后，整个 Claude Code 会话同步到手机上。

- 手机和终端**完全同步**，可交替操作
- 代码始终在电脑本地跑，手机只是遥控器
- 文件系统、MCP 服务器、项目配置全部在本地，安全可靠

---

## 九、/export — 导出对话为 Markdown

输入 `/export`，当前整段对话导出为 Markdown 文件。

**适用场景**：
- 和 Claude 讨论了半小时架构方案，导出保存作为后续的详细 context
- 导出后扔给 Codex 等其他工具做交叉审查

## 十、常用快捷键

| 快捷键 | 功能 | 备注 |
|--------|------|------|
| `Ctrl+V` | 直接粘贴截图 | Mac 用户注意是 Ctrl 不是 Cmd |
| `Ctrl+J` / `Option+Enter` | 输入框换行 | Mac 两种都可以 |
| `Ctrl+R` | 搜索历史 prompt | 快速找到之前用过的指令 |
| `Ctrl+U` | 删除整行输入 | 清空当前输入框 |
| `Esc Esc` | /rewind 回退 | 连按两次 Esc |

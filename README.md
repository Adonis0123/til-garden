# TIL Garden

Adonis 的数字花园 — 记录每天学到的小知识。

## 在线访问

👉 [https://adonis-til.mintlify.app/introduction](https://adonis-til.mintlify.app/introduction)

## 技术栈

基于 [Mintlify](https://mintlify.com/) 构建的文档站点。

## 本地开发

```bash
# 安装 Mintlify CLI
pnpm add -g mintlify

# 启动本地预览
mintlify dev
```

## 目录结构

```
├── ai/          # AI 相关文章
├── frontend/    # 前端相关文章
├── docs.json    # Mintlify 配置（导航、主题等）
└── introduction.mdx  # 首页
```
